# Fruit Ripeness App

A web application for determining the ripeness of fruits from uploaded images.

The system uses a convolutional neural network based on EfficientNetB0.

## Supported fruits

- Banana
- Mango
- Strawberry

## Technologies

### Frontend

- React
- Vite
- JavaScript
- CSS

### Backend

- Python
- TensorFlow
- Keras
- Flask or FastAPI

## Main functions

- Image uploading
- Image validation
- Fruit recognition
- Ripeness classification
- Displaying recommendations

## Project structure

- `frontend` — web interface
- `backend` — server and model inference
